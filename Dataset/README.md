The folder stores Brightness Temperatures (BTs) measured by Advanced Baseline Imager (ABI) on board the GOES-17 (137.2Â° west) geostationary satellite. 
The files are stored in .mat formats. 
Each file contains:
- two matrices of latitude (deg) and longitude (deg)
- a 3D-tensor of BTs. The third dimension identifies the wavelenght ($\lambda$):
    - 8.50 $\mu m$
    - 9.70 $\mu m$
    - 10.30 $\mu m$
    - 11.20 $\mu m$
    - 12.30 $\mu m$
    - 13.30 $\mu m$
