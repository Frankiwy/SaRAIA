The folder stores Brightness Temperatures (BTs) measured by ABI (Advenced Baseline Imager) on board the GOES-17 (137.2Â° west) geostationary satellite. 
The files are stored in .mat formats. 
Each file contains:
- two matrices of latitude (deg) and longitude (deg)
- a 3D-tensor of BTs. The third dimension identifies the wavelenght ($\lambda$):
    - 8.50 um
    - 9.70 um
    - 10.30 um
    - 11.20 um
    - 12.30 um
    - 13.30 um
